# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in Advanced
Trigonometry Calculator (ATC), please report it privately to the project
maintainer instead of opening a public issue with exploit details.

When reporting, include:

- affected version or branch;
- steps to reproduce;
- expected and actual behavior;
- any files or commands involved;
- possible impact, if known.

Please avoid sharing proof-of-concept material publicly until the maintainer
has had time to review the report.

## Supported Versions

The supported version policy is currently limited to the **current maintained
branch** of the repository.

Older releases may still be useful, but no broad security maintenance promise
is made for historical versions unless explicitly stated in release notes.

## Scope

ATC is a local command-line application. Security reports may include issues
such as unsafe file handling, command execution risks, memory safety problems,
or behavior that could unexpectedly affect user data.

This policy does not promise a fixed response time, but good-faith reports are
appreciated.
