# Code of Conduct

Advanced Trigonometry Calculator is an open-source project. Contributors and
users are expected to keep discussion respectful, practical, and focused on the
work.

## Expected Behavior

- Be respectful of different levels of experience.
- Discuss ideas and code without personal attacks.
- Give clear, constructive feedback.
- Assume good faith when possible.
- Keep bug reports and technical discussions focused on reproducible facts.

## Unacceptable Behavior

- Harassment, threats, or personal insults.
- Discriminatory language or behavior.
- Repeatedly derailing technical discussions.
- Publishing private information without permission.
- Intentionally misleading reports or hostile contributions.

## Maintainer Responsibility

The maintainer may moderate issues, pull requests, or discussions to keep the
project useful and respectful.

This code of conduct is intentionally simple. The goal is a healthy project
space where people can improve ATC together.
